## Contexto del proyecto

SiCROA es un sistema interno de gestión para el Hangar Oficial del Gobierno del Estado de Oaxaca (DGHOGE). El sistema centraliza el seguimiento de operaciones aeronáuticas, personal técnico, documentos oficiales y programas de capacitación.

## El desafío

El cliente manejaba toda la información en hojas de cálculo y archivos dispersos. El objetivo fue construir un sistema web moderno que:

- Centralizara la gestión de personal aeronáutico
- Digitalizara el control de documentos y certificaciones
- Permitiera registrar y dar seguimiento a capacitaciones
- Siguiera la normativa oficial de la DGAC

## Solución técnica

La arquitectura usa **Feature-Sliced Design (FSD)** para escalar el frontend de forma predecible. Cada módulo es autónomo y sigue el mismo patrón.

### Stack elegido

- **React + TypeScript** — type safety en todo el pipeline
- **TanStack Query** — cache inteligente de datos del servidor
- **React Hook Form + Zod** — validación en formularios multi-paso
- **FormData + multipart** — carga de archivos oficiales

## Resultados

El sistema pasó de 0 a producción en 4 meses, reemplazando el flujo manual y reduciendo errores de captura en el registro de personal.
